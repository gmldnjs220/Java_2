        ```sql
        int [] n = {1,2,3,4,5};

        for (int k : n){ //반복될 때마다 k는 n[0], n[1],...n[4]로 번갈아 설정
        	sum += k;
        }
        ```

        - 함수형 언어에서는  함수 조합으로 복잡한 작업을 수행하며, 상태 변경 대신에 데이터를 변환하는 방식으로 프로그램을 작성함
        - 재귀, 고차함수, 순수 함수 등의 개념을 중요시하며, 병렬 처리와 높은 수준의 추상화를 지원
        - 함수 지향 언어는 객체 지향 언어의 특성을 그대로 가져가되, 함수 지향적인 기능을 추가한 것
        - ex) Python, Kotlin, Haskell, Lisp, Scala
        - 객체 지향 언어의 복잡함을 보완하기 위해 나옴
